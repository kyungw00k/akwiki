# 설치 가이드

akwiki를 설치하는 방법입니다.

## Homebrew

macOS와 Linux에서 가장 간편한 방법입니다.

```
brew install kyungw00k/cli/akwiki
```

## Go

Go 1.22 이상이 설치되어 있다면:

```
go install github.com/kyungw00k/akwiki/cmd/akwiki@latest
```

## Shell script

curl로 바로 설치할 수 있습니다.

```
curl -sSfL https://kyungw00k.github.io/akwiki/install.sh | sh
```

## 바이너리 직접 다운로드

[Releases](https://github.com/kyungw00k/akwiki/releases) 페이지에서 OS/Arch에 맞는 바이너리를 다운로드하세요.

지원 플랫폼:

- linux/amd64, linux/arm64
- darwin/amd64, darwin/arm64
- windows/amd64, windows/arm64

## 첫 위키 만들기

설치 후 바로 시작해보세요:

```
akwiki init my-wiki
cd my-wiki
akwiki dev
```

브라우저에서 `http://localhost:3000`을 열면 위키가 보입니다.

자세한 문법은 [[마크다운 문법]]을 참고하세요.
