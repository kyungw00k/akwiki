# 테마 시스템

akwiki의 테마는 3단계로 커스터마이징할 수 있습니다.

## 1단계: 기본 테마

`akwiki init`만 실행하면 wiki.g15e.com과 동일한 디자인이 적용됩니다.

- OKLCh 색공간 기반 색상 시스템
- 라이트/다크 모드 자동 전환
- Noto Serif KR 제목 + 시스템 산세리프 본문
- 3컬럼 Grid 레이아웃 (데스크톱)
- 반응형 디자인 (모바일 단일 컬럼)

## 2단계: config.yml

`.akwiki/config.yml`에서 색상, 폰트, 레이아웃을 변경합니다.

```yaml
theme:
  colors:
    background: "#1a1a2e"
    text: "#eaeaea"
    link: "#e94560"
    accent: "#0f3460"
  fonts:
    heading: "Georgia, serif"
    body: "Verdana, sans-serif"
  layout:
    toc: true
    backlinks: false
    related: true
    search: true
```

설정값은 CSS 변수로 주입되므로 템플릿을 수정할 필요가 없습니다.

## 3단계: 템플릿 오버라이드

`.akwiki/theme/` 디렉토리에 파일을 두면 해당 부분만 교체됩니다.

```
.akwiki/theme/
├── templates/
│   ├── page.html           ← 전체 레이아웃
│   └── partials/
│       ├── header.html
│       ├── toc.html
│       ├── backlinks.html
│       ├── related.html
│       ├── search.html
│       └── footer.html
└── static/
    └── style.css           ← 추가 CSS
```

파일이 없으면 내장 기본 테마가 사용됩니다.

## 디자인 디테일

원본 wiki.g15e.com에서 가져온 디테일들:

| 요소 | 구현 |
|------|------|
| 링크 밑줄 | 두께 0.5px, offset 0.25em |
| 외부 링크 | ↗ 아이콘 (superscript) |
| 강조(bold) | font-weight 대신 하이라이트 배경 |
| 인용문 | 세리프 폰트 + 좌측 녹색 border |
| 구분선(HR) | `- - - § - - -` 텍스트 |
| TOC | 데스크톱에서 sticky 고정 |
| 한국어 | word-break: keep-all |

[[마크다운 문법]]에서 각 요소의 사용법을 확인하세요.
