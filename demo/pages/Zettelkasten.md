# Zettelkasten

Zettelkasten(독일어: 메모 상자)은 사회학자 Niklas Luhmann이 사용한 지식 관리 방법론입니다.

## 핵심 원칙

1. **원자성** — 하나의 노트에 하나의 생각
2. **연결** — 모든 노트는 다른 노트와 연결
3. **자기 언어** — 원문을 그대로 옮기지 않고 자기 말로 재구성
4. **고유 ID** — 각 노트를 고유하게 식별

## akwiki에서의 적용

akwiki로 Zettelkasten을 실천할 수 있습��다:

- **하나의 .md 파일 = 하나의 제텔** — 원자성 보장
- **[[위키링크]]** — `[[]]` 문법으로 노트 간 연결
- **백링크** — 역방향 참조 자동 추적
- **[[관련 콘텐츠]]** — TF-IDF로 미처 연결하지 못한 관련 노트 발견
- **aliases** — 같은 개념의 다른 이름으로도 링크 가능

## Luhmann의 시스템

Luhmann은 약 90,000장의 카드를 작성하여 70권 이상의 책과 400편 이상의 논문을 발표했습니다. 그의 생산성의 비밀은 카드 간 연결에 있었습니다.

> "나는 혼자 생각하지 않는다. 대부분은 Zettelkasten 안에서 일어난다."
> — Niklas Luhmann
