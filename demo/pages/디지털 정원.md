# 디지털 정원

디지털 정원(Digital Garden)은 완성된 글을 발행하는 블로그와 달리, 생각의 씨앗을 심고 가꾸는 공간입니다.

## 블로그와의 차이

| | 블로그 | 디지털 정원 |
|---|------|----------|
| 구조 | 시간순 (타임라인) | 주제별 (네트워크) |
| 상태 | 완성된 글 발행 | 성장하는 글 |
| 연결 | 카테고리/태그 | 양방향 링크 |
| 독자 | 외부 독자 | 자기 자신 + 공유 |

## akwiki와 디지털 정원

akwiki의 기능은 디지털 정원에 적합합니다:

- **[[위키링크]]** — 생각 간 연결을 자연스럽게 표현
- **백링크** — "이 생각을 참조하는 다른 생각들" 자동 추적
- **[[관련 콘��츠]]** — 미처 연결하지 못한 관련 글 발견
- **private 페이지** — 아직 공개하기 이른 씨앗은 숨기기
- **Git 기반 날짜** — 글의 성장 이력이 자연스럽게 기록
