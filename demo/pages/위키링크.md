# 위키링크

위키링크는 akwiki의 핵심 기능입니다. `[[` `]]` 문법으로 페이지 간 연결을 만들면, 양방향 링크가 자동으로 추적됩니다.

## 기본 문법

```markdown
[[페이지 이름]]
```

이렇게 쓰면 해당 페이지로의 링크가 생성됩니다. 예: [[Home]]

## 표시 텍스트

```markdown
[[페이지 이름|다른 텍스트]]
```

링크 대상은 같지만 표시되는 텍스트를 바꿀 수 있습니다. 예: [[Home|홈 페이지]]

## 별칭(Aliases)

frontmatter에 `aliases`를 등록하면 해당 이름으로도 링크할 수 있습니다.

```yaml
---
aliases:
  - 스펙다운
  - spec-down
---
```

## 백링크

위키링크를 통해 양방향 참조가 자동으로 추적됩니다. 이 페이지를 참조하는 다른 페이지들이 하단에 표시됩니다.

[[관련 콘텐츠]] 기능과 함께 사용하면 지식 네트워크를 구축할 수 있습니다.

## 링크 상태

| 상태 | 표시 |
|------|------|
| 존재하는 페이지 | 일반 링크 (파란색, 밑줄) |
| Private 페이지 | 비활성 링크 (회색, 점선 밑줄) |
| 존재하지 않는 페이지 | Missing 링크 (회색, 점선) |

Private 페이지로의 링크는 `#private-link`로 치환되어 내용이 노출되지 않습니다.
