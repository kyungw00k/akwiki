# 검색 기능

akwiki는 서버 없이 클라이언트 사이드에서 동작하는 검색을 제공합니다.

## 사용법

**Ctrl+K** (macOS: **Cmd+K**)를 누르면 검색창이 열립니다. **Esc**로 닫습니다.

## 검색 대상

빌드 시 생성되는 `search-index.json`에는 각 페이지의 다음 필드가 포함됩니다:

| 필드 | 가중치 | 설명 |
|------|--------|------|
| title | 높음 | 영문 제목 |
| titleKo | 높음 | 한국어 제목 |
| brief | 보통 | 첫 문단 요약 |
| tags | 보통 | 태그 |
| aliases | 보통 | 별칭 |

제목 매칭이 가장 높은 우선순위를 가집니다.

## 구현

- 외부 라이브러리 없이 순수 JavaScript로 구현
- 인덱스는 첫 검색 시 lazy 로드
- 결과는 유사도 점수 순으로 상위 10개 표시

[[테마 시스템]]에서 검색 UI를 커스터마이징할 수 있습니다.
