# 관련 콘텐츠

akwiki는 빌드 시점에 TF-IDF 텍스트 유사도를 계산하여 관련 콘텐츠를 자동 추천합니다.

## 작동 방식

1. 모든 페이지의 본문 텍스트를 토큰화합니다
2. TF-IDF(Term Frequency-Inverse Document Frequency) 벡터를 생성합니다
3. 코사인 유사도로 페이지 간 유사도를 계산합니다
4. 유사도 상위 10개를 타입별로 그룹핑하여 표시합니다

## 타입별 분류

관련 콘텐츠는 페이지의 `type` 필드에 따라 분류됩니다:

| 타입 | 표시 라벨 |
|------|----------|
| Article | 관련 글 |
| Book | 관련 책 |
| ScholarlyArticle | 관련 논문 |
| Journal | 관련 일지 |
